# DockAdmin — Administrator Guide

**Top-level invocation prefix:** `/dockadmin`

This section documents administrative commands and interactions for managing Dockhound on a guild.

## Contents

- `/dockadmin settings view` — download current guild settings as JSON
- `/dockadmin settings config-update` — upload a GuildConfig JSON to update server configuration
- `/dockadmin settings guild-update` — update stored guild name/tag
- `/dockadmin settings logs-per-message` — show log events for a message id
- `/dockadmin verify info` — post verification info message and save reference
- `/dockadmin verify restrict` — set verification access mode

Each page describes parameters, permissions, examples, and notes about persistence and audit actions.
