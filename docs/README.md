# Dockhound Manual

**Overview**

This manual contains two main sections:

- **User** commands and interactions available to regular server members (invoked under the `/dock` root).
- **DockAdmin** administrative commands and configuration actions available to guild administrators (invoked under the `/dockadmin` root).

Each section contains per-command pages describing the command purpose, parameters, permission requirements, examples, and notes about interactive components (buttons / select menus).
